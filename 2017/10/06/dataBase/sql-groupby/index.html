四、having子句

过滤分组。例：求平均工资大于2000的部门：

where 与having区别：

  • where后不可以使用分组函数，having可以，当不使用分组函数时，两者相同
  • having 先分组，再过滤
  • where先过滤，再分组，效率高，使分组记录数大大降低
  • SQL优化：通用时使用where

五、order by子句
