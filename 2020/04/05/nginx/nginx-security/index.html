介绍

常见的恶意行为：爬虫行为和恶意抓取、资源盗用

  • 基础防盗链功能，目的不让恶意用户能轻易的爬取网站对外数据
  • secure_link_module对数据安全性提高加密验证和失效性，适合核心重要数据
  • access_module对后台、部分用户服务的数据提供IP防控
